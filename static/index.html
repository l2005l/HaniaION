OPEN SOURCE SOFTWARE PROJECT

HaniaION

GNSS Klobuchar Data API & RAAM Converter

A Python and FastAPI utility that retrieves the latest BRDC navigation data from NASA CDDIS, extracts Klobuchar ionospheric coefficients, and converts them for GNSS and RAAM software.

DEVELOPER PLATFORM

Software Development Features

BRDC Retrieval

Automatically locates and downloads the latest GNSS navigation file from NASA CDDIS Earthdata.

RINEX Parsing

Parses the navigation header and extracts the Klobuchar Alpha and Beta ionospheric coefficients.

RAAM Conversion

Converts scientific coefficients into signed integer values suitable for RAAM navigation software.

REST Backend

Built with Python and FastAPI, packaged with Docker, and deployed as a reproducible web service.
